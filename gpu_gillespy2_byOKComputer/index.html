快速开始示例

创建模型

from gpu_gillespie import DimerizationModel

# 创建二聚化模型
model = DimerizationModel(
    k_dimerization=0.1,
    k_dissociation=0.01,
    initial_A=1000,
    initial_A2=0
)

# 运行GPU加速模拟
results = model.run_simulation(
    time_span=(0, 100),
    n_trajectories=10000
)

性能分析

# 查看性能统计
