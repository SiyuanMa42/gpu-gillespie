GPUGillespieSimulator

高性能GPU加速的Gillespie随机模拟器主类。提供简单易用的接口进行并行随机模拟计算。

初始化

GPUGillespieSimulator
__init__( species_names: List[str], reaction_names: List[str], stoichiometry: np.ndarray, rate_constants: Union[List[float], np.ndarray], initial_conditions: Dict[str, int] = None )

参数说明

参数 类型 说明
species_names List[str] 化学物种名称列表
reaction_names List[str] 反应名称列表
stoichiometry np.ndarray 化学计量矩阵 [n_reactions, n_species]
rate_constants Union[List[float], np.ndarray] 反应速率常数
initial_conditions Dict[str, int] 初始条件（可选，默认为全0）

主要方法

Dict
run_simulation( time_span: Union[List[float], Tuple[float, float]], n_timepoints: int = 101, n_trajectories: int = 1000, use_tau_leaping: bool = False, seed: Optional[int] = None )

运行GPU加速的Gillespie随机模拟。 

# 创建模拟器
simulator = GPUGillespieSimulator(
    species_names=['A', 'B'],
    reaction_names=['production', 'degradation'],
    stoichiometry=np.array([[1, 0], [0, -1]]),
    rate_constants=[1.0, 0.1],
    initial_conditions={'A': 100, 'B': 50}
)

# 运行模拟
results = simulator.run_simulation(
    time_span=(0, 100),
    n_timepoints=101,
    n_trajectories=5000,
    use_tau_leaping=False
)

ParallelSimulator

高级并行模拟管理器，支持参数扫描、集合模拟和敏感性分析。

主要方法

Dict
parameter_sweep( parameter_name: str, parameter_values: List[float], fixed_parameters: Dict[str, float] = None, time_span: Tuple[float, float] = (0, 100), n_timepoints: int = 101, n_trajectories_per_point: int = 1000, n_workers: int = 4 )

执行参数扫描分析，探索参数空间对系统行为的影响。 

# 创建并行模拟器
parallel_sim = ParallelSimulator(simulator)

# 参数扫描
sweep_results = parallel_sim.parameter_sweep(
    parameter_name='k_cat',
    parameter_values=[0.1, 0.5, 1.0, 2.0, 5.0],
    n_trajectories_per_point=2000,
    n_workers=4
)
Dict
ensemble_simulation( parameter_sets: List[Dict[str, float]], time_span: Tuple[float, float] = (0, 100), n_timepoints: int = 101, n_trajectories_per_set: int = 1000, n_workers: int = 4 )

运行集合模拟，测试多个参数组合的效果。

DimerizationModel

简单的二聚化模型：2A ⇌ A₂。用于演示基本的可逆反应动力学。

DimerizationModel
__init__( k_dimerization: float = 0.1, k_dissociation: float = 0.01, initial_A: int = 1000, initial_A2: int = 0 ) 
# 创建二聚化模型
model = DimerizationModel(
    k_dimerization=0.1,
    k_dissociation=0.01,
    initial_A=1000,
    initial_A2=0
)

# 获取理论平衡常数
k_eq = model.get_equilibrium_constant()
print(f"平衡常数: {k_eq:.3f}")

# 运行模拟
results = model.run_simulation(time_span=(0, 100), n_trajectories=1000)

EnzymeKineticsModel

米氏酶动力学模型：E + S ⇌ ES → E + P。经典的酶催化反应机制。

EnzymeKineticsModel
__init__( k_binding: float = 1.0, k_dissociation: float = 0.1, k_cat: float = 0.5, initial_E: int = 100, initial_S: int = 1000, initial_ES: int = 0, initial_P: int = 0 ) 
# 创建酶动力学模型
enzyme_model = EnzymeKineticsModel(
    k_binding=1.0,
    k_dissociation=0.1,
    k_cat=0.5,
    initial_E=100,
    initial_S=1000
)

# 计算米氏常数
km = enzyme_model.get_michaelis_constant()
print(f"米氏常数 Km: {km:.3f}")

# 计算最大速度
vmax = enzyme_model.get_max_velocity(100)
print(f"最大速度 Vmax: {vmax:.1f}")

快速开始

1. 安装

pip install gpu-gillespie

2. 基本使用

from gpu_gillespie import DimerizationModel

# 创建模型
model = DimerizationModel(
    k_dimerization=0.1,
    k_dissociation=0.01,
    initial_A=1000,
    initial_A2=0
)

# 运行模拟
results = model.run_simulation(
    time_span=(0, 100),
    n_trajectories=10000
)

# 查看结果
print(f"执行时间: {results['performance_stats']['execution_time']:.2f}s")
print(f"加速比: {results['performance_stats']['speedup_factor']:.1f}x")

3. 高级分析

# 分析结果
analysis = model.simulator.analyze_results(results)

# 创建可视化
fig = model.simulator.plot_trajectories(results, n_sample_trajectories=50)

# 导出数据
analyzer = model.simulator.SimulationAnalyzer(results)
analyzer.export_to_csv('simulation_results.csv')
